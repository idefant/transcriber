# Словарь

## Назначение

Словарь хранит пользовательские слова, имена, термины и написания, которые нужно подсказывать модели распознавания речи.

## Просмотр

Экран словаря показывает список слов в виде тегов. Если слов нет, отображается пустое состояние.

Слова сортируются для отображения с учетом русской локали.

## Добавление слова

Пользователь вводит слово или фразу в поле и добавляет ее кнопкой или клавишей Enter.

Правила:

- пробелы в начале и конце удаляются;
- пустое значение не добавляется;
- дубликаты не добавляются;
- сравнение дубликатов чувствительно к регистру, поэтому слова, различающиеся только регистром, считаются разными словами и могут сосуществовать в словаре;
- после успешного добавления поле ввода очищается;
- пока идет сохранение, ввод и удаление ограничены состоянием загрузки.

## Удаление слова

Пользователь удаляет слово закрытием соответствующего тега.

Правила:

- удаление чувствительно к регистру и убирает только слово с точным совпадением;
- после удаления список сохраняется и заново отображается;
- при ошибке пользователь видит сообщение.

## Использование словаря в диктовке

При запуске распознавания речи приложение загружает словарь, объединяет слова через запятую и подставляет получившуюся строку в prompt Speech-to-Text.

Словарь не выполняет жесткую замену текста после распознавания. Он работает как подсказка модели, поэтому итоговый результат зависит от поведения выбранной модели и качества аудио.
