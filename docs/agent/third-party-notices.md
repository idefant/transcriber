# Сторонние уведомления

`THIRD_PARTY_NOTICES.md` содержит уведомления только для сторонних ресурсов, которые приложение распространяет как отдельные файлы: бинарные библиотеки, модели, веса моделей и другие assets из `bundle.resources`.

Не перечисляйте в этом файле внутренние зависимости Rust, JavaScript или других экосистем вручную только потому, что они используются при сборке или скомпилированы в приложение. Полный учёт лицензий всех зависимостей — отдельная задача, которую нужно решать автоматизированно и единообразно для всего проекта.

Для каждого отдельно распространяемого ресурса:

- добавьте требуемые copyright и текст лицензии;
- включите `THIRD_PARTY_NOTICES.md` в тот же дистрибутив, что и ресурс;
- при добавлении ресурса в `bundle.resources` проверьте, что notice также попадёт в установщик;
- не удаляйте существующее уведомление без явного решения пользователя о лицензионном риске.
